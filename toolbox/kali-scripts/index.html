Kali Linux Setup Scripts

Transform your Kali Linux desktop into a modern environment with ready-to-use bash scripts. These scripts automate the installation and configuration of Alacritty terminal, tmux, Arc-Dark theme, and custom bash prompt.

Requirements: Kali Linux with XFCE • Internet connection and sudo privileges • 10-30 minutes runtime

Get the complete repository from GitHub

⚠️ WARNING

These scripts perform full system updates and modify core packages. Create a system snapshot before running. Always review scripts before copying and executing them. Use at your own risk.

Setup Script

Main script that transforms your Kali desktop with modern tools: Alacritty terminal, tmux, Arc-Dark theme, custom wallpaper, and enhanced bash prompt. 

git clone https://github.com/AskDatDude/setup_kali_env.git
cd setup_kali_env
bash setup_kali_env.sh

Restore Script

Completely reverts all changes made by the setup script, removing packages, configurations, and restoring original theme and wallpaper. 

bash ~/setup_kali_env/restore_kali_env.sh
